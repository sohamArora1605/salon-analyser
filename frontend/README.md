
# Frontend

Vite React dashboard for the Salon Analytics Platform. It includes KPI cards, EDA charts, inventory views, no-show risk summaries, and a simple no-show predictor.

## Setup

```powershell
npm install
Copy-Item .env.example .env.local
npm run dev
```

Set `VITE_API_BASE_URL` to the deployed backend URL when publishing to Vercel.
