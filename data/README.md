# Data Folder

`raw/` contains the salon CSV files copied from the workspace root.

For this prototype, we treat all files as if they belong to one hypothetical salon:

- Appointments: `Future Bookings (All Clients)0.csv`
- Cancellations and no-show-style events: `Client Cancellations0.csv`, `No-Show Report0.csv`, `salon_noshow_data.csv`
- Services: `Service Listing0.csv`
- Retail products: `Product Listing (Retail)0.csv`
- Sales transactions: `Receipt Transactions0.csv`
- ML seed data: `hair_salon_no_show_wrangled_df.csv`
